## ğŸ“¦ Installation & Setup

### Development Setup

```bash
# Clone the repository
git clone https://github.com/M-Atif-Latif/freetoolz.git

# Navigate to project directory
cd freetoolz

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Production Deployment

The site is deployed on a VPS with Nginx. To deploy updates:

```bash
# Build the project
npm run build

# Deploy to VPS (update with your server details)
scp -r dist/* root@freetoolz.cloud:/var/www/freetoolz/

# Fix permissions on server
ssh root@freetoolz.cloud "sudo chown -R www-data:www-data /var/www/freetoolz && sudo chmod -R 755 /var/www/freetoolz && sudo systemctl reload nginx"
```

Or use the quick deploy script:

```bash
.\quick-deploy-to-vps.ps1 -ServerIP your-server-ip -Username root
```
